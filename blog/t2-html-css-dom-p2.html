What is grid based design?

Grids are like the glue that holds a design together.The purpose of a grid is to establish a set of guidelines for how elements should be positioned within a layout.
Whats all the hype about responsive webpages?
Responsive design isn’t just about making your page display properly on any device, you also have to make it function well – and that means it has to be good at accepting input in a world where desktops have touchscreens and phones have keyboards.
